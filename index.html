📘 How to Play

  1. กรอกชื่อผู้เล่น แล้วกดปุ่ม Start
  2. เลือกหัวข้อและประโยคที่ต้องการฝึก
  3. กด ฟังเสียง เพื่อฟังประโยคตัวอย่าง
  4. กด เริ่มพูด แล้วอ่านตามประโยค
  5. ระบบจะแสดงคะแนน Pronunciation Assessment เป็นเปอร์เซ็นต์
  6. คะแนนตั้งแต่ 50% ขึ้นไปจะมีเสียง “Complete”
  1. Enter the player's name and press Start.
  2. Choose a topic and a practice sentence.
  3. Press Listen to hear the model sentence.
  4. Press Start speaking and read the sentence aloud.
  5. The game displays a Pronunciation Assessment percentage.
  6. A score of 50% or higher plays the word “Complete”.

ตั้งค่า Firebase ครั้งแรก
  1. สร้าง Firebase Project และเพิ่ม Web App
  2. เปิด Authentication → Anonymous
  3. สร้าง Cloud Firestore แล้วคัดลอก Rules ที่อยู่ในหน้าต่างนี้
  4. คัดลอก firebaseConfig มาวางในช่องด้านบน
Firebase Config ใช้ในเว็บได้และไม่ใช่รหัสผ่าน แต่ต้องป้องกันข้อมูลด้วย Firestore Rules
